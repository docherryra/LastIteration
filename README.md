# Last Iteration

> 2025 게임소프트웨어공학 1조 프로젝트

**Unity 기반 멀티플레이어 FPS(1인칭 슈팅) 게임**

## 개요

Last Iteration은 Photon Fusion 네트워크 프레임워크를 활용한 실시간 멀티플레이어 FPS 게임입니다. 최대 4명의 플레이어가 팀 데스매치 형식으로 경쟁하며, 킬 수에 따라 무기가 자동으로 업그레이드되는 독특한 시스템을 갖추고 있습니다.

## 기술 스택

| 구분 | 기술 |
|------|------|
| 엔진 | Unity 2022.3.62f1 |
| 네트워크 | Photon Fusion |
| UI | TextMesh Pro |
| 언어 | C# |

## 게임 특징

### 게임 모드
- **경기 시간**: 5분 (300초)
- **승리 조건**: 10킬 달성 또는 시간 종료 시 최다 킬
- **최대 플레이어**: 4명
- **최소 시작 인원**: 2명

### 무기 시스템 (킬 기반 업그레이드)

| 무기 | 획득 조건 | 데미지 | 탄창 | 특징 |
|------|-----------|--------|------|------|
| **라이플** | 시작 무기 | 20 | 10발 | 단발 정밀 사격 |
| **샷건** | 4킬 이상 | 10×8발 | 10발 | 산탄 스프레드 (15°) |
| **피스톨** | 8킬 이상 | 20 | 10발 | 빠른 연사 |

### 플레이어 조작

| 키 | 동작 |
|----|------|
| `W/A/S/D` | 이동 |
| `Shift` | 달리기 (6 m/s) |
| `Ctrl` | 앉기 (2 m/s) |
| `Space` | 점프 |
| `좌클릭` | 발사 |
| `우클릭` | 조준 (ADS) |
| `R` | 재장전 |
| `Tab` | 스코어보드 |

### 전투 시스템
- **체력**: 100 HP
- **리스폰 시간**: 3초
- **탄약 표시**: 색상 코드 (빨강: 빈 탄창 / 노랑: 33% 이하 / 흰색: 정상)
- **조준 시스템**: 화면 중앙 기준 발사

## 프로젝트 구조

```
Assets/
├── Animations/              # 캐릭터 애니메이션
├── Material/                # 게임 머티리얼
├── Photon/                  # Photon Fusion 네트워크
├── Resources/               # 런타임 리소스
├── Scenes/                  # 게임 씬
│   ├── BootScene           # 부팅 씬
│   ├── AuthScene           # 인증 씬
│   ├── menuScene           # 메인 메뉴
│   ├── SampleScene         # 게임 플레이
│   └── EndScene            # 결과 화면
├── Scripts/                 # 게임 스크립트
│   ├── Api/                # 백엔드 API 연동
│   ├── Auth/               # 인증 시스템
│   ├── Camera/             # 카메라 제어
│   ├── EndScene/           # 종료 UI
│   ├── Network/            # 네트워크 로직
│   └── Player/             # 플레이어 메카닉
└── TextMesh Pro/           # UI 텍스트 시스템
```

## 핵심 스크립트

### 게임 관리
| 스크립트 | 역할 |
|----------|------|
| `GameManager` | 매치 타이머, 승리 조건 판정 |
| `NetworkRunnerHandler` | Photon 세션 관리 |
| `PlayerSpawner` | 플레이어 생성 및 입력 처리 |
| `Spawner` | 리스폰 위치 선정 |

### 플레이어
| 스크립트 | 역할 |
|----------|------|
| `PlayerState` | HP, 킬, 데스, 리스폰 관리 |
| `PlayerMovement` | 이동, 점프, 앉기 |
| `CameraController` | 1인칭 카메라 조작 |
| `WeaponManager` | 무기 전환 및 조준 |

### 무기
| 스크립트 | 역할 |
|----------|------|
| `Gun` | 라이플 발사 로직 |
| `Shotgun` | 산탄총 스프레드 발사 |
| `Bullet` | 탄환 물리 및 충돌 |

### UI
| 스크립트 | 역할 |
|----------|------|
| `HUDKillDeathUI` | K/D 및 리스폰 타이머 |
| `CombatLogUI` | 킬 로그 피드 |
| `ScoreboardUI` | 순위표 |
| `HPBar` | 체력바 |

## 게임 플로우

```
BootScene → AuthScene → menuScene → 매칭 → SampleScene → EndScene
              (로그인)    (메뉴)     (Photon)   (게임)      (결과)
```

## 네트워크 아키텍처

- **호스트/클라이언트 모델**: 호스트가 데미지, 킬, 리스폰 판정
- **동기화 데이터**:
  - `PlayerState`: 닉네임, HP, 킬, 데스
  - `PlayerMovement`: 위치, 회전, 속도
  - `WeaponManager`: 현재 무기, 발사 사운드


## 빌드 및 실행

1. Unity Hub에서 프로젝트 열기 (Unity 2022.3.62f1)
2. Photon App ID 설정 (Assets/Photon/Fusion/Resources)
3. `File → Build Settings`에서 빌드
4. 또는 에디터에서 `Play` 버튼으로 테스트
